## Dependencies
#### [Server]
* express
* socket.io
* dockerode
* utf8

#### [Client]
* monaco-editor
* socket.io-client
* xterm.js

## Setup
- Download and Install Docker Desktop Client
- Pull ubuntu container

```bash
docker pull ubuntu
```
- Clone the repo and install the dependencies 

```bash
git clone https://github.com/JayGoyal96/PlayGround.git
cd client
npm install
cd ..
cd server
npm install
```
- To run the app, write

```bash
cd client
npm start
cd ..
cd server
npm start
```
